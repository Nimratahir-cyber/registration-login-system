# Registration-Login-System
This is a simple console-based registration and login system developed in C++. It uses file handling to store user credentials and allows users to register and log in securely through a menu-driven interface.

#Requirements
This project requires a C++11 or later compliant compiler (tested on C++14).
#Features
##Included libraries
<iostream> for standard input output
<windows.h> for Windows API functions such as keyboard button clicks.
<sstream> for extracting words from string
<fstream> for file handling

##How it works
###Registraction
enter username and password of at least 8 characters
the record is stored in a text file("thisfile.txt")
###Login
user enters credentials
if match-> login successfull
if not match->error message is shown
